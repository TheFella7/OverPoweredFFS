# Dragon Ball: Beyond Legacy

A 2D tile-based Dragon Ball inspired game built with HTML5, CSS3, and JavaScript. Play as a Saiyan, Human, or Namekian and collect all 7 Dragon Balls!

## Features
- Choose from 3 unique character classes
- Tile-based exploration
- Real-time combat system
- Dragon Ball collection
- Full mobile and desktop support
- Responsive design for all devices

## How to Play
1. Enter your character name
2. Choose your class (Saiyan, Human, or Namekian)
3. Explore the world using arrow keys or WASD
4. Collect all 7 Dragon Balls
5. Defeat enemies and power up!

## Controls
- **Arrow Keys / WASD**: Move character
- **Space / F**: Fire Ki Blast
- **C**: Charge Ki
- **R**: Toggle Flight
- **ESC**: Pause Game
- **Touch Controls**: On-screen D-pad and action buttons

## Classes
- **Saiyan**: High power, grows stronger
- **Human**: Balanced, masters techniques
- **Namekian**: Regenerates health, high defense

## Deployment
Simply open `index.html` in any modern web browser. No installation required!

## Technologies Used
- HTML5 Canvas
- CSS3 with Flexbox/Grid
- Vanilla JavaScript
- Responsive Design
